# PM原型拾取器 (PM Prototype Picker) 🚀

一款专为产品经理（PM）、UI/UX 设计师和前端开发设计的 Chrome 浏览器插件。帮助用户从任意网页中快速、精准地提取 UI 组件的结构（HTML）、样式（CSS）、定位坐标（多点物理边界）及 XPath 路径，并支持在精致的沙盒预览舱中进行实时渲染与交互。

---

## ✨ 核心特性

- **🎨 极极致的毛玻璃 UI 设计**
  - 基于 React + TypeScript + Vite 构建。
  - 面板与弹窗采用高端 Glassmorphism 毛玻璃暗黑科技风格，辅以 Outfit/JetBrains Mono 字体与幻彩发光粒子背景，提供极佳的交互与视觉质感。
- **🎯 自由元素拾取与智能提取**
  - 支持页面 Hover 实时高亮与点击锁定。
  - 智能样式提取算法：自动合并并过滤冗余样式，对根节点强制保留颜色、字体等继承排版属性，确保在沙盒中完美还原原网页视觉效果。
- **📐 3x3 五点物理坐标系统**
  - **尺寸大小**：紧凑展示 `W × H` 像素物理大小。
  - **坐标锚点**：模拟设计工具（如 Figma/Sketch）的对齐逻辑，直观展示元素的 **左上 (TL)**、**右上 (TR)**、**居中 (Center)**、**左下 (BL)**、**右下 (BR)** 五个核心特征点的绝对像素坐标值。
- **📦 沙盒预览舱 (Sandbox)**
  - 使用 `iframe sandbox` 安全地隔离渲染被提取的组件。
  - 沙盒内与模态框内的所有文本均支持鼠标划拉框选、高亮与复制。
  - 底部提供一键复制 HTML、复制 CSS、复制 XPath，并伴有绿勾动效 Toast 提示。
- **⏳ 历史记录与源溯源**
  - 自动记录本地历史提取卡片，支持悬浮卡片微升阻尼动效。
  - 历史缩略图采用纯白底色，完美承托各种透明 SVG/PNG 组件。
  - 记录“捕获时间”并保存“源网页地址（白底黑字高易读卡片）”，方便随时点击回访。
- **🖱️ 阻尼长按拖拽与折叠**
  - 独立 Header 拖拽手柄，长按即可自由移动面板；双击折叠为精致的 locator 悬浮小球。
  - 支持拖拽边界碰撞检测，防误触算法智能区分“点击折叠”与“长拖拽定位”。

## 🛠️ 安装与部署指南

### 1. 本地编译与打包

确保您的电脑上已安装 [Node.js](https://nodejs.org/)。

```bash
# 1. 克隆项目到本地
git clone https://github.com/yoyo-halo/PMPrototype.git
cd PMPrototype

# 2. 安装依赖包
npm install

# 3. 运行生产环境编译
npm run build
```

编译完成后，项目根目录下会生成一个 `dist/` 文件夹（包含已编译的 `background/index.js`、`content/index.js`、`manifest.json` 及相关图标与样式资源）。

### 2. 加载到 Chrome 浏览器

1. 打开 Chrome 浏览器，在地址栏输入：`chrome://extensions/` 并回车。
2. 在右上角开启 **“开发者模式” (Developer mode)**。
3. 点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
4. 在弹出的文件选择框中，选择本项目根目录下的 **`dist` 文件夹**。
5. 成功加载后，点击浏览器右上角拼图图标将 **PM原型拾取器** 固定到工具栏。

### 3. 开始使用

1. 打开任意您想要提取 UI 元素的网页，并**刷新该网页**以让插件脚本注入生效。
2. 点击浏览器工具栏的插件图标，或使用快捷键 `Ctrl + Shift + E` 唤醒右下角悬浮控制面板。
3. 点击 **“自由拾取”** 即可在网页中 Hover 目标元素，点击后在弹出的历史卡片中双击或点击查看详情。

## 📂 项目结构

```text
PMPrototype/
├── public/                 # 静态资源及清单文件
│   ├── icons/              # 插件图标 (16px, 48px, 128px)
│   └── manifest.json       # Chrome 扩展配置文件 (MV3)
├── src/
│   ├── background/         # Service Worker (后台消息中枢与截图裁剪服务)
│   ├── content/            # 注入脚本 (核心选择器、样式提取引擎、DOM分组器)
│   ├── popup/              # React 应用主界面 (悬浮面板、历史卡片、沙盒预览模态框)
│   └── utils/              # 通用存储工具方法
├── package.json            # 脚本及依赖配置
├── tsconfig.json           # TypeScript 配置
└── vite.config.ts          # Vite 构建配置 (多入口打包)
```
